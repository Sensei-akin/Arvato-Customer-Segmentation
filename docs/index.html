exception customer_segmentation.models.DataSplitsSizeException[source]¶

Customized exception just to clarify “ValueError” from sklearn.model_selection.train_test_split behavior when the test or validation size is not correct and none is equal to zero

exception customer_segmentation.models.DataSplitsUnitException[source]¶

Custom exception to make sklearn.model_selection.train_test_split only works with the float unit
